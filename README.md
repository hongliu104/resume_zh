#### Thanks to

- [posquit0/Awesome-CV](https://github.com/posquit0/Awesome-CV)

#### Usage

```bash
# configure environment
sudo apt-get install texlive-full
# edit resume.tex and cv/*.tex as your wish
# generate resume.pdf
make
```
